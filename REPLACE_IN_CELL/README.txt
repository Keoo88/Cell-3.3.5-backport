Эта папка содержит только изменённые/новые файлы для Cell.

Как использовать:
1. Скопируйте Cell/Cell.toc поверх вашего Cell/Cell.toc (замена).
2. Скопируйте папку Cell/Libs/ClassicAPI в ваш Cell/Libs/ (новая папка).
3. Отключите standalone-аддон !!!ClassicAPI в списке аддонов.
4. Запустите игру и проверьте.

ClassicAPI встроен в Cell как библиотека (по принципу TSM compat),
урезан до модулей, которые реально использует Cell.
